CamlChess

CS 3110 Final Project (Fall 2022)

Command Line Chess

This was a group project for the final of the Cornell core CS course CS 3110: Functional Programming and Data Structures. The final result is a fully functional, command-line chess game with features such as a graveyard, move history, and even the option to face an AI opponent.
